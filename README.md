# MyPod — Галерея фотографий iPod

Статический HTML-сайт-галерея фотографий iPod в разных локациях (Сиэтл, Бирмингем).
Учебный проект на чистом HTML/CSS (без JavaScript).

## Файлы

- `index.html` — Главная страница с миниатюрами
- `html/*.html` — 6 страниц-деталей с полноразмерными фото
- `photos/` — 9 фотографий
- `thumbnails/` — 6 миниатюр
- `logo/` — Логотип

## Запуск

Открой `index.html` в браузере, нажимай на миниатюры для просмотра.
